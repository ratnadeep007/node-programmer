## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn or

### Installation

1. Clone the repository
```bash
git clone [repository-url]
cd node-programmer
```

2. Install dependencies
```bash
npm install
# or
yarn install
```

3. Start the development server
```bash
npm run dev
# or
yarn dev
```

The application will be available at `http://localhost:5173`
